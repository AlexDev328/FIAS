# FIAS
Веб приложение для работы локального использования ФИАС.
Позволяет выгрузить базу и развернуть локально с некоторыми плюшками)
---
# План работ на ближайшее время следующий:
-----
- [ ] Распаковка и загрузка файлов БД ФИАСА
- [ ] Реализация автообновления данных и автоматизация первичной выгрузки
- [ ] Поиск объектов
- [ ] Полнотекстовый поиск по адресам
